PORTABLE SETUP INSTRUCTIONS

Files you should have in this folder:
1. app.py
2. best.pt (Your trained model)
3. requirements.txt

Step-by-Step Guide to Run on a New Computer:

1. Prerequisite:
   - Ensure Python is installed on the computer.

2. Setup:
   - Copy this folder to the new computer.
   - Open the folder.
   - Type 'cmd' in the address bar and press Enter to open a terminal.

3. Install Dependencies (First time only):
   - Run the command: pip install -r requirements.txt

4. Run the App:
   - Run the command: streamlit run app.py

Note: I have updated app.py to automatically look for 'best.pt' in this folder.
